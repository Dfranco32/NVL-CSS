Prueba de de aplicación de estilos - CSS (Tipo 1)

CSS sirve para dar estilo y diseño a las páginas web — cambiar de tipo de letra, color, tamaño y espaciado del contenido, mostrar múltiples columnas, añadir animaciones y otras funcionalidades.

En este archivo haremos una mezcla de los trs modos de aplicar estilos a un HTML

En este ejemplo, el código HTML es un fichero llamado, NVL-CSS-P3a4.html y los estilos están en el llamado stylesP3a4.css.
