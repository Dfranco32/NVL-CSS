Prueba de de aplicación de estilos - CSS (Tipo 2)

CSS sirve para dar estilo y diseño a las páginas web — cambiar de tipo de letra, color, tamaño y espaciado del contenido, mostrar múltiples columnas, añadir animaciones y otras funcionalidades.

Hay tres maneras distintas de aplicar CSS a un documento HTML.

En este caso no enlazamos a ningún fichero, ya que los estilos se encuentran embebidos (inmersos) dentro de un contenedor especial HTML en la etiqueta <style>…</style>, dentro de la sección <head> del documento.
