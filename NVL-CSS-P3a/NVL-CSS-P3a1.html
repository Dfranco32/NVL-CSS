Prueba de de aplicación de estilos - CSS (Tipo 1)

CSS sirve para dar estilo y diseño a las páginas web — cambiar de tipo de letra, color, tamaño y espaciado del contenido, mostrar múltiples columnas, añadir animaciones y otras funcionalidades.

Hay tres maneras distintas de aplicar CSS a un documento HTML.

En este caso utilizaremos la etiqueta <link>. Esta etiqueta nos permite incluir referencias a otros ficheros que contienen lógica perteneciente a la página.

En este ejemplo, el código HTML es un fichero llamado, NVL-CSS-P3a1.html y los estilos están en el llamado stylesP3a1.css.
